## 🆕 Updates — What Changed (Latest Version)

This is the **v2 redesign** of the portfolio. Here is what i updated:

### Visual Changes
- **New colour palette** — moved from plain dark blue to a deeper navy + warm peach (`#FFA586`) + crimson red accent. It look way more intentional now
- **New fonts** — using `DM Sans` for body and `Space Mono` for code-style labels. Much better personality
- **Background grid overlay** — subtle dot/line grid in the hero section, give it a more techy feel
- **Glowing blobs** — soft radial blur elements in background for depth (inspired by modern design trend)

### Layout Changes
- Added **sticky navigation bar** that become transparent-glass when you scroll
- Added **floating stat widgets** in hero section (TryHackMe points, active paths etc)
- Completely **redesigned skills section** — now have icon, progress bar, and hover animation
- **Project cards** redesigned with tags, icons and "Featured" badge
- **Certification cards** now show status (In Progress / Active) clearly
- **Contact section** redesigned as interactive cards that slide on hover

### Code Changes
- Added `IntersectionObserver` for scroll animations — cards fade in when they enter viewport
- Skill progress bars animate when you scroll to them
- Navbar get glass blur effect on scroll
